# Local-Film-rating
A local film rating web app

## Features

- Local database using Json (*you don't need encryption for your personal and local reviews don't you ?*).
- Search films using TMDB database and API.
- Review it with 4 categories : Plot, Visuals, Music and Acting, and add a comment !
- See your reviews and sort them as you wish.
- Select your personal favorits.

> Obviously vibe coded but what a result in 1 hour !
